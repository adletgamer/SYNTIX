## ⚡ The Vision: Hybrid Identity
In the current digital landscape, health data is either trapped in institutional silos or exposed in vulnerable databases. **SYNTIX** bridges the gap between biological reality and digital sovereignty. 

We envision a world where your genetic and clinical markers are not just "records," but active, verifiable, and private assets that power your digital identity.

## 🎬 The Problem: The Data Cemetery
Traditional health data management suffers from:
- **Zero Privacy:** Data is often sold or leaked, losing its biological confidentiality.
- **Fragmented Silos:** Clinical biomarkers are scattered across different providers.
- **Lack of Verification:** No way to prove health status without revealing the underlying sensitive data.

## 🛡️ Core Pillars

### 1. Confidential Compute (FHE)
Powered by **Inco Network**, SYNTIX performs computations on encrypted health data. Your biomarkers never leave their encrypted state during analysis.

### 2. Zero-Knowledge Proofs (ZKP)
Utilizing **Risc Zero**, we generate succinct proofs of health insights (e.g., "Verified non-smoker" or "Optimal PGx Profile") without disclosing the raw genomic sequences.

### 3. Clinical-Grade Metadata
Integrated with **Medplum (FHIR)**, SYNTIX ensures that the data flowing through the protocol adheres to global clinical standards, making it interoperable with modern healthcare systems.

## 🛠 Features
- **Neural Overlay Hero:** A high-fidelity visual interface reflecting the "data death" and "biological rebirth" through scroll-based interactions.
- **PGx Interaction Checker:** A real-time engine to check genomic compatibility with various medications privately.
- **Confidential Vault:** Secure storage for 100+ clinical biomarkers.
- **ZKP Ticker:** Real-time verification status of on-chain health proofs.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation
```bash
git clone https://github.com/adletgamer/SYNTIX.git
cd SYNTIX
npm install
```

### Run Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the terminal.

## 🏗 Tech Stack
- **Frontend:** Next.js 14, React, Tailwind CSS
- **Animations:** Framer Motion
- **Privacy Layer:** Inco (Confidential Compute)
- **Verification:** Risc Zero (ZKP)
- **Data Standards:** Medplum / FHIR
- **Blockchain:** Conflux eSpace (Target Mainnet)
